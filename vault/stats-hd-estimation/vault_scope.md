# Vault: stats-hd-estimation

## Covers
- Covariance matrix estimation (shrinkage, graphical lasso, random matrix)
- Copula modeling (parametric, semiparametric, tail dependence)
- Bootstrap and simulation methods for validating estimators
- Dimensionality constraints (p >> n, small-sample estimation)

## Does Not Cover
- Bayesian estimation (future vault if project work demands)
- Robust estimation / M-estimators (future vault)
- General machine learning (future vault)
- Time series forecasting (out of scope)

## Boundary Cases
- Regularization: covered when applied to estimation; not covered as general optimization
- Simulation: covered when validating estimators; not covered as standalone Monte Carlo
